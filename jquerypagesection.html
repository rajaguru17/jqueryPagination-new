Pagination design
Pagination Attributes
S.no Option Name Action Description
1. totalItems optional (default value 1) The total number of items in the page should be sent so that it will calculate the total pages along with the itemsPerPage.
2. itemsPerPage optional (default value 1) The total number of items in one page should be sent so that it will calculate the total pages along with the totalItems.
3. currentPage optional (default value 1) This will set the page to the current value.
Pagination call function
			$(function() {
				$(selector).pagesection({
					totalItems :1,
					itemsPerPage : 1,
					currentPage : 1,
					onPageChange : function(page){
						//will get the current page
					}
			 });
            });
